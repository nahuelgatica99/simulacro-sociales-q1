# 🌍 Simulador de Ciencias Sociales - Q1

Este es un simulador interactivo diseñado para el repaso de los contenidos de **Ciencias Sociales** correspondientes al primer trimestre. La aplicación está optimizada para estudiantes, con un enfoque visual y pedagógico basado en el libro de texto de abril de 2026.

## 🚀 Funcionalidades

- **Carga Dinámica:** Selecciona aleatoriamente 30 preguntas de un banco de 100 en cada partida.
- **Feedback Educativo:** Cada respuesta incorrecta muestra una explicación detallada basada en los textos escolares.
- **Sistema de Puntos y Rangos:** Motiva al estudiante a mejorar su puntaje para alcanzar el rango de "Maestro".
- **Accesibilidad:** Botón de lectura en voz alta para facilitar la comprensión de las consignas.
- **Interfaz Temática:** Diseño adaptado con una paleta de colores roja/bermellón específica para el área de Sociales.

## 📖 Temas Evaluados

El banco de preguntas cubre los siguientes ejes temáticos:
1.  **Ambientes y Recursos Naturales:** (Conceptos de ambiente, valoración social, Barrancas de Belgrano).
2.  **Manejo de Recursos:** Renovables, No Renovables y Perpetuos (Sustentabilidad).
3.  **Actores Sociales:** El rol del Estado, ONGs, empresas y ciudadanos.
4.  **Problemas Ambientales:** Escalas local y global (Deforestación, contaminación, cambio climático).
5.  **Riesgos y Amenazas:** Amenazas internas (sismos, volcanes) y atmosféricas (tornados, sudestadas).
6.  **Vulnerabilidad:** Prevención de desastres y casos históricos (Terremoto de San Juan 1944).

## 🛠️ Instalación y Uso

Para ejecutar este proyecto localmente o en GitHub Pages:

1.  Clona el repositorio o descarga los archivos `index.html`, `style.css` y `script.js`.
2.  Asegúrate de que los tres archivos estén en la misma carpeta.
3.  Abre el archivo `index.html` en tu navegador.

## 🎨 Personalización Visual

La distribución de la pantalla se mantiene fiel al diseño original, utilizando las siguientes variables de color:
* **Bermellón:** `#e34234` (Bordes y elementos principales).
* **Rojo Oscuro:** `#8b0000` (Títulos y botones de navegación).
* **Coral:** `#ff6b6b` (Detalles y hovers).
